# Reports Directory

Outputs generated by `make postprocess` land here. The key entry point is
`RESULTS_MANIFEST.json`, which indexes per-model summaries, head rankings, OV
reports, and H5/H6 consolidated tables. Use `notebooks/results_summary.ipynb`
to browse interactively.

Common files:

- `*_summary_table.csv` — logit diff means & seed counts per probe
- `*_trio_percentiles.json` — suppressor trio statistics
- `{model}_{probe}_head_ranking.csv` (+ top/bottom slices)
- `ov_report_*.json` — token projections for notable heads
- `h5_pairs_ranking.csv`, `h5_triplets_ranking.csv`, `h6_layer_ranking.csv`

Regenerate:

```
make postprocess
python scripts/verify_manifest.py  # optional sanity check
```
